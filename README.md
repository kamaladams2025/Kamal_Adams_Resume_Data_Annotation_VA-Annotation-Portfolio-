# Kamal Adams — Data Annotation & QA Portfolio

This repository documents applied, self-directed practice in multilingual data annotation, structured-dataset QA, and style-guide-driven content review — built during a gap-year certification program (2025–2026) ahead of remote data annotation and virtual assistant roles.

Each folder below is a self-contained work sample with its own README explaining the methodology, the schema/tagging conventions used, and (where applicable) a documented QA pass showing how errors were identified and corrected. The goal is not just to show finished output, but to make the *process* — the judgment calls, edge cases, and validation steps — visible to a reviewer.
